# Changelog

All notable changes to Case Switcher will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-18

### Added

- Initial release of Case Switcher.
- 10 Intelligent Text Transformation Modes (Title Case, URL Slug, etc.).
- Multi-language support (English, Portuguese, Spanish, French, German, Italian).
- Acronym Immunity system for technical documentation.
- Customizable Global Keyboard Shortcuts (`Alt+Shift+0-9`).
- Context Menu integration.
- Privacy-first architecture (offline, no data collection).
