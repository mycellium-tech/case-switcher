# Privacy Policy

Case Switcher is committed to protecting your privacy.

## Data Collection

This extension does not collect any data.

## What This Extension Does

- Transforms selected text on web pages
- Stores your preferences locally using browser storage
- All processing happens entirely on your device

## What This Extension Does Not Do

- Send any data to external servers
- Track your browsing activity
- Access any personal information
- Make any network requests

## Permissions Explained

| Permission | Purpose |
|------------|---------|
| contextMenus | Display transformation options in the right-click menu |
| storage | Save your language and mode preferences locally |
| scripting | Inject content script to transform text on pages |
| all_urls | Allow the extension to work on any website |
